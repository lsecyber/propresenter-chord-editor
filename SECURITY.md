# Security Policy

This tool reads and writes local ProPresenter files. Treat presentations and chord charts as sensitive by default because they may contain copyrighted material, private library metadata, or organization-specific information.

## Reporting a Vulnerability

Please do not publish sensitive examples or private presentation files in public issues. If this repository is hosted on GitHub, use private vulnerability reporting when available. Otherwise, open a minimal public issue that describes the problem without attaching private files, and the maintainer can arrange a safer way to receive details.

## Supported Versions

Until the project has tagged releases, security fixes target the main branch.
